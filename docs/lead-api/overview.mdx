## Purpose

This integration pushes Sales Partner Lead records into **Salesforce production** via the Salesforce REST API. Partners use a dedicated custom object (`Sales_Partner_Lead__c`) that is isolated from internal Lead data. It supports three operations:

1. **Discover** — query the Sales Partner Lead schema so your integration adapts to custom fields automatically.
2. **Create** — insert a new Sales Partner Lead.
3. **Upsert** — insert-or-update a Sales Partner Lead by an External ID field, preventing duplicates and making retries safe.

## Prerequisites

Before these endpoints will work, a Lumina Salesforce administrator must supply a client id and secret

<Warning>
  These endpoints are part of the standard Salesforce REST API. They will not work without a Connected App and an authorized integration user in the Lumina Solar org.
</Warning>

## High-level flow

```text
1. Authenticate                 →  obtain an OAuth access token
2. Get your User ID             →  GET  /services/data/v60.0/chatter/users/me
3. Describe Sales Partner Lead  →  GET  /services/data/v60.0/sobjects/Sales_Partner_Lead__c/describe
4. Create Sales Partner Lead    →  POST /services/data/v60.0/sobjects/Sales_Partner_Lead__c
   — or —
   Upsert Sales Partner Lead    →  PATCH /services/data/v60.0/sobjects/Sales_Partner_Lead__c/{ExternalIdField}/{value}
```

## Base URL

All endpoints use the Lumina Solar Salesforce production instance:

```text
https://luminasolar.my.salesforce.com
```

<Tip>
  Use the **upsert** pattern whenever possible. It is idempotent — safe to retry without creating duplicates.
</Tip>

## Required fields

Every Sales Partner Lead submitted through this API must include: **First_Name__c**, **Last_Name__c**, **Email__c**, **Phone__c**, **Street__c**, **City__c**, **State__c**, **Postal_Code__c**, **UTM_Source__c** (must be set to `Partner`), and **OwnerId** (Salesforce User or Queue ID that should own the record).

See [Create a Sales Partner Lead](/docs/lead-api/create-lead) for the full required fields table and examples.
